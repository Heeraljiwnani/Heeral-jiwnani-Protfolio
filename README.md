## ✨ Key Features

-   🌌 **Interactive 3D Background**: A performant, particles-based 3D background using **Three.js**.
-   💎 **Glassmorphism UI**: Modern, translucent card designs with backdrop blurs.
-   🌓 **Dark & Light Mode**: Seamless theme toggle with persistent state.
-   📱 **Fully Responsive**: Mobile-first architecture ensuring a great experience on all devices.

## 🏃‍♂️ Getting Started

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/heeraljiwnani/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
